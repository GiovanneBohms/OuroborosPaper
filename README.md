# Ouroboros

Ouroboros é uma extensão para o cliente da Steam que automatiza completamente a coleta, análise e gestão de dados do mercado da Steam. Com foco na negociação de itens virtuais como cartas, skins e itens. A extensão permite otimizar a compra e venda de ativos dentro da plataforma, ajudando a identificar oportunidades lucrativas com base em critérios personalizáveis.

## Funcionalidades

- **Rastreamento de Itens**: Encontra itens com liquidez alvo e faz análise preditiva para identificar se será feito a compra, se sim, ele fornece ao usuário preço de compra e venda
- **Análise do inventário**: Verifica se algum item do inventário terá queda no lucro, se sim é sinalizado o cancelamento da ordem.
